# FlappyBird_DQN
This program uses the Deep Reinforcement Learning to play the Flappy Bird, the origin paper of this method is ["Playing Atari with Deep Reinforcement Learning"](https://www.cs.toronto.edu/~vmnih/docs/dqn.pdf) 

## Pre-Requisite
tensorflow 1.1.0
python 2.7.x
openai gym
openai gym_ple
opencv-python
